## [1.0.0] - 2026-03-25

### Added
- Initial release of xservice-arkts
- Forked from https://www.npmjs.com/package/xservice (version 0.0.11)
- Support for generating Flutter plugin code for multiple platforms:
  - Android (Java)
  - iOS (Objective-C)
  - Dart
  - ArkTS/OHOS (HarmonyOS) - New feature
- Support for both JSON and YAML input formats
- Command-line interface with various options
- Template-based code generation

### Changed
- Updated repository URL to https://github.com/Miosas/xservice-arkts.git
