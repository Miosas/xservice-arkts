## [1.0.0] - 2026-03-25

### 新增
- xservice-arkts 初始版本发布
- 从 https://www.npmjs.com/package/xservice (版本 0.0.11) 分叉而来
- 支持为多个平台生成 Flutter 插件代码：
  - Android (Java)
  - iOS (Objective-C)
  - Dart
  - ArkTS/OHOS (HarmonyOS) - 新特性
- 支持 JSON 和 YAML 两种输入格式
- 命令行界面，支持多种选项
- 基于模板的代码生成

### 变更
- 更新仓库 URL 为 https://github.com/Miosas/xservice-arkts.git

- 无
